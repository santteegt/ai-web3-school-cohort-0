## Definition

ERC-8004 is a draft standard for agent identity, reputation, and verification registry. It breaks down an agent's identity, reputation, and validation into composable registries, allowing different applications to discover and evaluate agents using the same format.

## Key Points

- Does not guarantee claimed capabilities are real — capabilities must still be supported by validation and trust models
- Provides a public carrying layer for identity, feedback, and verification signals
- Different applications build their own filtering and ranking rules on top of this foundation
- Does not attempt to make trust a single centralized score
- Concerns: Sybil attacks, review farming, evaluator trustworthiness, task type differences must still be addressed
- Complements [[erc-8183]]: ERC-8004 covers identity/reputation; ERC-8183 covers task/payment/escrow lifecycle
