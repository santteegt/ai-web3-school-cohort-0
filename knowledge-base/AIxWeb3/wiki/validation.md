## Definition

Validation is the verification process for agent capabilities or task results. It should record test data, version, time, and validator. Distinguish between capability validation (agent is likely capable of a task type) and task result validation (specific delivery is qualified).

## Key Points

- Types: automated tests, benchmarks, manual audits, review by another set of agents, on-chain proofs, or TEE attestations
- For agent marketplaces: make validation results a queryable history, not just a certification badge
- Capability validation ≠ task result validation — both are needed in different contexts
- Validators themselves need reputation to be trusted as evaluators
