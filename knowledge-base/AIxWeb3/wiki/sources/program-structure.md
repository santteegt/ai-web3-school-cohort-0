## Summary

This source describes the four-week program structure of the AI × Web3 School bootcamp. Week 1 builds shared language across AI and Web3 foundations through hands-on execution. Week 2 enters the intersection problem space with track selection. Week 3 deepens practice and kicks off the hackathon. Week 4 is the hackathon sprint and demo showcase. The program emphasizes execution over theory: wallet creation, testnet transactions, contract deployment, and agent workflow practice are all required in Week 1.

## Notable Points

- Week 1 execution chain: LLMs → prompts → workflows → agents → tool use → AI coding → wallets → signatures → transactions → gas → smart contracts → testnets.
- Track options for Week 2: Agentic Commerce/Payment, Dev Tooling, AI Security/Privacy, AI × Governance/Coordination, Open Track.
- Final submission requires: project name, track, demo link, repository, README, video, and on-chain verification materials (contract addresses, transaction hashes).
