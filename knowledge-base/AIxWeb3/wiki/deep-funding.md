## Definition

Deep Funding focuses on the allocation of funds for public goods and complex contributions. AI can help organize project impacts, contribution relationships, and review materials — but fund allocation still requires human value judgment, diverse reviews, and transparent rules.

## Key Points

- Challenge: impact is not linear — a low-level library used by many projects, a coordinator making multiple teams successful, a documentation project lowering barriers
- AI builds evidence packages: project dependency graphs, use cases, contributor networks, historical funding, user feedback
- Final allocation must retain diverse reviews, public reasoning, and appeal mechanisms
- Models should not become black-box grantors
