## Definition

Explorer Context is visual on-chain evidence provided by block explorers. In AI products, providing an explorer link is more reliable than just saying "transaction successful" — users can verify the link themselves and developers can troubleshoot errors.

## Key Points

- Answers: did a transaction succeed, is a contract verified, is source code public, were events emitted, did token transfers occur
- Provides checkable entry points for both users and Agents
- Explorer links create accountability: interpretations backed by a link can be verified and challenged
- Suitable for beginner-level context collection
