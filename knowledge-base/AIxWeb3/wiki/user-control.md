## Definition

User Control means users can view, modify, and revoke the permissions and data usage of an AI agent. If users cannot see which model an agent is using, what memories it has saved, what tool permissions it holds, or how much it can spend, there is no real control.

## Key Points

- Control panel minimum: connected wallets, session keys, authorized limits, enabled tools, memory toggles, data export entries, emergency stop button
- These must not be hidden in advanced settings — they are core user interface elements
- "Can the user see and change the system state" is the test for meaningful user control
- Directly connected to sovereignty: control is only real when mechanisms exist to act on it
