# DeFi (Decentralized Finance)

## Definition

Decentralized Finance (DeFi) is an ecosystem of financial protocols built on public blockchains that operate without intermediaries. Lending, trading, yield generation, and stablecoin systems are governed by smart contracts, open to anyone with a wallet, and auditable by anyone who can read the chain.

## Key Points

- **Permissionless**: no KYC, no application, no approval — any wallet can interact with any DeFi protocol
- **Non-custodial**: users retain control of assets; contracts hold them only during transactions (contrast with CEX custody)
- **Composability**: protocols can be stacked — borrow from Aave, swap on Uniswap, deposit into Yearn — in a single transaction
- **Core verticals**: token trading (AMMs), lending/borrowing, stablecoins, yield aggregators, derivatives
- **Key risks**: smart contract bugs, oracle manipulation, liquidation cascades, MEV, impermanent loss, regulatory risk
- **Gas dependence**: DeFi profitability depends on gas costs; high L1 gas makes small-size trades unprofitable → L2 DeFi
- **AI × Web3 relevance**: AI agents as automated portfolio managers, yield optimizers, and arbitrageurs — all operating via session keys and smart accounts
