## Definition

ERC-7702 is an Ethereum Improvement Proposal that allows standard externally owned accounts (EOAs) to temporarily gain smart-account-like capabilities. Instead of migrating to a full smart account, an EOA can delegate its execution to a smart contract for a specific transaction or session, enabling programmable logic without giving up the base key.

## Key Points

- EOAs can temporarily act as smart accounts by pointing to a contract implementation
- Does not require migrating to a new address or abandoning the existing private key
- Enables programmable features (batch transactions, gas abstraction, session keys) on demand for standard wallets
- Complements [[erc-4337]] — ERC-4337 creates new smart accounts; ERC-7702 upgrades existing EOAs
- Reduces friction for AI agents needing temporary elevated permissions on a user's existing wallet
