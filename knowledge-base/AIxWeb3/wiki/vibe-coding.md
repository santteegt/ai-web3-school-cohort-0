## Definition

Vibe coding is a rapid prototyping practice where developers use AI coding tools (Claude Code, Cursor, Codex) to generate, iterate, and deploy working code quickly — prioritizing speed of execution and learning-by-doing over upfront design.

## Key Points

- In the AI × Web3 School program, vibe coding with Hermes Agent is a required Week 1 hands-on exercise
- AI coding tools can rapidly generate boilerplate, explain unfamiliar libraries, and accelerate prototyping — but architectural decisions, test design, and code review cannot be delegated to them
- The output of vibe coding sessions must still be validated: the faster you generate code, the more important validation becomes
- Vibe coding is particularly powerful for: first contact with a new framework, exploring API surface, generating repetitive scaffolding, and understanding unfamiliar codebases
- Connects to the broader AI Foundations principle: model output is a candidate result, not a fact
