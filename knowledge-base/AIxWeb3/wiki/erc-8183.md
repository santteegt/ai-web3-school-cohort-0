## Definition

ERC-8183 is a draft standard around agentic commerce that focuses on how agent tasks, payments, escrow, delivery, and verification form a unified process. It advances agent commerce from "just sending some money" to a structured transaction model where tasks, states, proof, settlement, and disputes are all system-understandable.

## Key Points

- Focus: the task lifecycle as a state machine — not just token transfers
- Complements [[erc-8004]]: ERC-8004 is about agent identity and reputation; ERC-8183 is about tasks, payments, escrow, and delivery
- A complete agent commerce system needs ideas from both
- Key abstraction: transactions in the agent economy are state transitions around the task lifecycle
