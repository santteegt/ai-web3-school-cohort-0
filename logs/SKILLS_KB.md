## Skills repos/marketplaces


- [Vercel's Open Agent Skills Ecosystem](https://skills.sh/)
  - [Github](https://github.com/vercel-labs/skills)
  - [Skill](https://skills.sh/vercel-labs/skills/find-skills) to discover and install skills

- [Claude Agents Skills](https://github.com/anthropics/skills)

- [Awesome Agent Skills](https://github.com/VoltAgent/awesome-agent-skills)
  - Community-curated list of agent skills with official links

- [Awesome Openclaw Skills](https://github.com/VoltAgent/awesome-openclaw-skills)
  - Filtered and categorized from the official OpenClaw Skills Registry.

- [Agent Skills for context engineering](https://github.com/muratcankoylan/Agent-Skills-for-Context-Engineering)
  - Marketplace: ClaudeCode plugin marketplace
  - A comprehensive collection of Skills for context engineering, multi-agent architectures, and production agent systems.

- [Stainpete skills](https://github.com/steipete/agent-scripts/)
  - MISC skills used by the creator of Openclaw for his agents

- [AgentGuard](https://github.com/numbergroup/AgentGuard)
  - Security framework that protects AI agents from prompt injection, command injection, and Unicode bypass attacks

- [Clawhub skills ](https://clawhub.ai/)
  - CLI tool to find and install skills in openclaw
  - Searchable via vector embeddings
  - Theres another library site called [Opencla skills library](https://open-claw-skills-library.vercel.app/) that seems to point to skills in this [Github](https://github.com/openclaw/skills) repository

- [SkillsMP](https://skillsmp.com/)
  - Agent Skills marketplace


## Web3 Tools/Skills

- [Coinbase Agentic Wallet](https://docs.cdp.coinbase.com/agentic-wallet/welcome)
  - Marketplace: Vercel skills
  - [Introduction blog post](https://www.coinbase.com/developer-platform/discover/launches/agentic-wallets)
  - [Agentic Wallet Skills repo](https://github.com/coinbase/agentic-wallet-skills)

- [Sign-in with Agent](https://siwa.id/)
  - Agent on-chain identity (via ERC-8004) and signed API requests (via ERC-8128) ([X post on how it works](https://x.com/builders_garden/status/2021628317003350069))
  - [Skill for agents](https://siwa.id/skill.md): agent reads and autonomously creates a wallet, registers onchain and authenticates

- [Openfort embedded wallets](https://www.openfort.io/)

- [Conway](https://docs.conway.tech/llms.txt)
- Give your agent the ability to create cloud sandboxes, pay for services through x402, run code, deploy applications, register domains, and expose services to the internet.
  - No Skills yet but there's the llms.txt that describes how to use conway-terminal CLI & MCP server

- [TrailOfBits plugin for smart contract security research, vulnerability detection, and audit workflows](https://github.com/trailofbits/skills)
  - Marketplace: ClaudeCode plugin marketplace
  - Each plugin can contain subagent + skills + resources

- [Solidity Auditor](https://github.com/pashov/skills)
  - A lightweight solidity auditor agent

- [Smart Contract Vulnerabilities Scan Skill](https://github.com/kadenzipfel/scv-scan/)

- [QuillShield Security Skills](https://github.com/quillai-network/qs_skills)
  - ClaudeCode plugin with skills for advanced smart contract security auditing

- [sc-auditor](https://github.com/Archethect/sc-auditor)
  - ClaudeCode plugin with skills for static analysis, Solodit findings search, Cyfrin checklist, and interactive Map-Hunt-Attack methodology.

- [Solidity Development Standards](https://github.com/Cyfrin/solskill)
  - Marketplace: Vercel skills

- [ETHSkills](https://ethskills.com/)
  - Marketplace: clawhub & ClaudeCode Plugin marketplace
  - Knowledge between AI agents and production Ethereum

- [Base Skills](https://github.com/base/skills)
  - Marketplace: Velcel Skills
  - Skills for building and interacting on Base L2 with Coinbase SDKs

- [Polygon Agent CLI + Skill](https://github.com/0xPolygon/polygon-agent-cli)
  - Marketplace: Vercel Skills
  - [Announcement blog post](https://polygon.technology/blog/polygon-launches-an-onchain-toolkit-built-for-the-agent-economy)
  - Skills for buildingn and interacting with Polygon chain

- [AgentMail](https://docs.agentmail.to/integrations/skills)
  - Skills for giving AI agents their own inboxes
  - [Skills repo](https://github.com/agentmail-to/agentmail-skills)
